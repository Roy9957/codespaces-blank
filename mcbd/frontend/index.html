❓ THE ENCHANTMENT TABLE

Frequently Asked

Is it free to join?
Yes — joining the Facebook group and Discord server is completely free, always.
Do I need a Java or Bedrock edition?
Both are welcome. Most of our community servers support cross-play between Java and Bedrock.
How do I report a rule-breaker?
Use the Report card above or ping a moderator directly in Discord — we respond fast.
Can I host my own event?
Absolutely — reach out to the Event Lead through a support ticket to get it scheduled.
